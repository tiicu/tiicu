# tiicu

> A beautiful, cross-platform emulation frontend heavily inspired by iiSU, built entirely in pure C++ with an initial focus on Nintendo Switch homebrew.

![Status](https://img.shields.io/badge/status-alpha-orange) ![Platform](https://img.shields.io/badge/platform-Switch%20%7C%20PC%20%7C%20Mac-red) ![License](https://img.shields.io/badge/license-Closed%20Source-lightgrey)

## What is tiicu?

**tiicu** is an emulation frontend that organizes your emulators and games in a visually stunning, intuitive interface. Built with pure C++, it's designed to be fully portable across platforms.

### Inspired by iiSU

This project draws inspiration from the iiSU frontend concept, aiming to recreate that nostalgic feeling of classic handhelds. With iiSU remaining closed source, we felt comfortable building **tiicu** as a closed-source alternative with primary focus on Nintendo Switch homebrew. 

## Features

### Currently Available

-   **Beautiful UI/UX** - Clean, minimal interface focused on playing games, not scrolling endless lists
-   **Automatic Game Organization** - Intelligent categorization based on No-Intro naming conventions
-   **Emulator Integration** - Initial cores already implemented and in testing
-   **Controller-First Design** - Built for handheld and controller gameplay (touch is not a priority)
- -   **Custom Home Screen** - Add game shortcuts

### In Development
-   **Enchanced Home Screen** - Add widgets, animated GIFs, and more
-   **Media Management** - Organize screenshots and animated GIFs for your home screen
-   **Online Asset Services** - Access curated game cards, backgrounds, and intro sounds from the tiicu community
-   **Asset Downloader** - Automatically fetch and apply community-curated assets

### Coming Soon

-   **RetroAchievements Integration** - Track your gaming accomplishments (planned once main cores are stable)

## Why tiicu?

-   **Pure C++ Architecture** - Fully portable across platforms with native performance
-   **Nintendo Switch at First** - Initial focus on delivering the best Switch homebrew experience
-   **Automated Organization** - No-Intro based sorting means less manual work
-   **Community Assets** - Curated, high-quality visual assets with online services

## Platform Support

**Current Focus:** Nintendo Switch (homebrew)

**Future Possibilities:** The pure C++ architecture allows tiicu to potentially run on other platforms. Future ports will be determined by community interest and demand through polls and feedback.

Possible platforms include desktop (Windows, macOS, Linux), mobile (Android, iOS), and other gaming devices.
